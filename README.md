# A library for interacting with the joystick.tv bot api
***This library is still in alpha and is not production ready***

| feature                                                          |                    status |
|:-----------------------------------------------------------------|--------------------------:|
| Recieve events from server                                       |     Done in `0.1.0-alpha` |
| Send events to server                                            | Planned for `0.2.0-alpha` |
| Send REST requests                                               | Planned for `0.3.0-alpha` |
| Record and playback events                                       | Planned for `0.4.0-alpha` |
| Generate rust structs for new stream events from recorded events | Planned for `0.5.0-alpha` |
| Production ready release                                         |       Planned for `0.6.0` |
